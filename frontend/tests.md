## Mobile

- generate-books
  - should generate 3 books to read when provided input
  - should generate 3 books to read when provided input, and they should differ if generated again - not enabled, flaky
  - should not generate particular books if they were liked before
  - should display an error when book field input is empty
  - should display an error when book field input is too short
  - should display an error when book field input is too long
  - should display error message when books are generated more than 6 times

---

- homepage
  - should display homepage

---

- like-books
  - should click like on a single generated book, and see it displayed in 'saved books'
  - should click like on two generated books, and see them displayed in 'saved books'
  - should click like on three generated books, and see them displayed in 'saved books'
  - should click like on generated books, and not see them displayed in 'saved books' for other user
  - should not see liked books if no books are liked after generating books

---

- log-in
  - should log in, when correct credentials are provided
  - should display error message, when log in credentials are incorrect

---

- log-out
  - should log out, when it is performed

---

- sign up
  - should sign up
  - Email
    - should display an error message when email is empty
    - should display an error message when email is too short
    - should display an error message when email is too long
    - should display an error message when email already exists
  - Password
    - should display an error message when password is empty
    - should display an error message when password is too short
    - should display an error message when password is too long
    - should display an error message when password is of right length but does not contain uppercase letter
    - should display an error message when password is of right length but does not contain lowercase letter
    - should display an error message when password is of right length but does not contain number
    - should display an error message when password is of right length but does not contain special symbol
  - Confirm password
    - should display an error message when confirm password is empty
    - should display an error message when confirm password does not match password
